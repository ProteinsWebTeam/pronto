Formatting help

Create a link to an external resource by putting the identifier of the resource, followed by a colon and the entry's accession, between square brackets, e.g. [pfam:PF00069].
The following resources are supported:

  • CATH / Gene3D: [cathgene3d:…]
  • CAZy: [cazy:…]
  • COG: [cog:…]
  • EC/INTENZ: [intenz:…]
  • Genome Properties: [genprop:…]
  • InterPro: [interpro:…]
  • PDBe: [pdbe:…]
  • Pfam: [pfam:…]
  • PIRSF: [pirsf:…]
  • PROSITE: [prosite:…]
  • SUPERFAMILY: [superfamily:…]
  • Swiss-Prot: [swissprot:…]
  • TIGRFAMs: [tigrfams:…]

Add a literature reference (citation) by putting cite:, followed by one or several PubMed IDs, in square brackets, e.g. [cite:11298287]. When adding several references, PubMed IDs should be separated by commas. When saving the annotation, PubMed IDs will be replaced by Publication IDs.
